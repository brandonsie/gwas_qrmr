Methods

Data

  • UK Biobank
    • ~488k participants
      • /n/groups/patel/uk_biobank/main_data_9512/
    • covariates: sex, age, 10-40 PC*
    • tested on three phenotypes:
      • BMI (#50-0.0)
      • height (#21001-0.0)
      • HbA1c (#30750-0.0)
      • /n/groups/patel/uk_biobank/main_data_34521/

Processing

  • mean linear regression with plink2 --glm (version 2020-07)
  • for quantile regression, choose quantiles tau <- c(0.1, 0.25, 0.5, 0.75, 0.9)
    • quantile linear regression with plink1.9 (version ?) + Rserve (1.8+) for custom R plugin using R package quantreg
    • quantile logistic regression with plink2 (version 2020-07)
  • meta-regression with R package metafor
    • \(y=\beta_0+\beta_1\tau+\beta_2(0.5-\tau)^2+e\)
    • \(p=\frac{p_{nominal}}{n_{snps}\cdot n_{meta}}=\frac{0.05}{800000\cdot3}\approx2\cdot10^{-8}\)

Runtime

On O2, per 1 cpu, with excess memory…

  • linear regression (42 covariates): \(\approx\) 0.002 sec / snp
  • logistic regression (42 covariates): \(\approx\) 0.02 sec / snp
  • quantile regression (12* covariates) \(\approx\) 96 sec / snp

plink 1.9 R plugin summary

Rplink <- function(PHENO, GENO, CLUSTER, COVAR){
  `%>%` <- magrittr::`%>%` 
  taus <- c(0.1, 0.25, 0.5, 0.75, 0.9) # quantiles to report
  qr_fun <- function(x){ # qr_fun is evaluated for each variant
    # initialize data frame for input into quantreg::rq
    qr_df <- data.frame(x = x, y = PHENO)[!is.na(x),] %>% (dplyr::bind_cols(COVAR))
    # Prepare regression eqation. `y ~ x + c1 + c2 + ... cp` for p covariates
    eqn <- as.formula(paste0("y ~ x + ", paste0(colnames(COVAR), collapse = "+")))
    # quantreg::rq for quantile regression
    q1 <- quantreg::rq(eqn, tau = taus, data = qr_df, method = "pfn")
    # quantreg::summary.rqs for standard error
    sum1 <- quantreg::summary.rqs(q1, se = "nid")
    #subset summary to relevant outputs (intercept and snp -- estimates and se)
    sum2 <- c(taus, sum1 %>% lapply(function(x) x$coefficients[1:2,] %>% as.numeric) %>% unlist)
    # plink expects output per variant as a vector of the form c(length(r), r)
    output_vector <- c(length(sum2), sum2)
  } # end qr_fun()
  # call qr_fun for each variant
  future::plan(future::multisession, workers = future::availableCores(""))
  q_out <- furrr::future_map(GENO %>% as.data.frame, qr_fun)
  return(q_out %>% unlist %>% matrix(nrow = 46))
} # end Rplink()
